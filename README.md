# 📡 ChurnRadar – Telecom Customer Churn Prediction

An AI-powered web application built using **Streamlit** that predicts whether a telecom customer is likely to churn or stay.
This project uses a **Random Forest Classifier** and provides an interactive UI with visual insights.

---

## 🚀 Live Demo

👉 https://adarsh07a-customer-churn-prediction-app-emcxjp.streamlit.app/
---

## 📌 Features

* 🎯 Predict customer churn in real-time
* 📊 Interactive dashboards (Gauge + Donut charts)
* 📈 Feature importance visualization
* 🧠 Machine Learning model (Random Forest)
* 🎨 Modern UI with custom styling
* ⚡ Fast and responsive Streamlit app

---

## 🧠 Model Details

* **Algorithm:** Random Forest Classifier
* **Features Used:** 40+ telecom customer attributes
* **Target:** Churn (Yes / No)

---

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/your-username/customer-churn.git
cd customer-churn
```

---

### 2️⃣ Create virtual environment (recommended)

```
python -m venv churn_env
```

Activate it:

**Windows**

```
churn_env\Scripts\activate
```

---

### 3️⃣ Install dependencies

```
pip install -r requirements.txt
```

---

### 4️⃣ Run the app

```
streamlit run app.py
```
